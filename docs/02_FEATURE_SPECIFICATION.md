## 1. 社員管理機能

### 1.1 社員一覧

**画面パス:** `/employees`

**機能概要:**
登録されている全社員を一覧表示し、検索・フィルタリングが可能です。

**表示項目:**
| 項目 | 説明 |
|------|------|
| アバター | 社員の顔写真（未設定時はイニシャル表示） |
| 氏名 | フルネーム |
| 所属チーム | 現在の所属チーム名 |
| 役職 | 現在の役職名 |
| TOP5資質 | ストレングスファインダーの上位5資質（バッジ表示） |

**検索・フィルタ:**
| フィルタ項目 | 説明 |
|--------------|------|
| キーワード検索 | 氏名、氏名（かな）、メールアドレスで部分一致検索 |
| チーム | 所属チームで絞り込み |
| 職種 | エンジニア、デザイナー等の職種で絞り込み |
| 在籍状況 | 在籍中/退職済みの切り替え |

**権限:**
| ロール | 閲覧 | 検索 |
|--------|------|------|
| 管理者 | ○ | ○ |
| マネージャー | ○ | ○ |
| 一般社員 | ○ | ○ |

---

### 1.2 社員詳細

**画面パス:** `/employees/:id`

**タブ構成:**

#### タブ1: 基本情報
| 項目 | 説明 |
|------|------|
| 氏名 | フルネーム（漢字） |
| 氏名（かな） | ふりがな |
| メールアドレス | 会社メールアドレス |
| 所属チーム | 現在の所属 |
| 役職 | 現在の役職 |
| 職種 | エンジニア、営業等 |
| 入社日 | YYYY年MM月DD日形式 |
| 権限 | 管理者/マネージャー/一般 |

#### タブ2: 傾向（得意・苦手）
- TOP5資質から推測される得意領域
- 下位資質から推測される苦手領域
- SPIワークスタイルとの関連分析

#### タブ3: この人の活かし方
- AIが生成するプロフィールサマリー
- 効果的な協働方法のアドバイス
- 成長に向けた提案

#### タブ4: ストレングスファインダー
- 34資質の順位一覧
- 4領域での分布グラフ（レーダーチャート）
- TOP5の詳細解説

#### タブ5: SPI分析
- 性格特性（Big Five）のグラフ
- ワークスタイル傾向のグラフ
- 適性スコア

#### タブ6: 評価履歴 ※権限制御あり
- 期別評価の一覧
- 評価グレード推移グラフ
- 強み・改善点のコメント

**権限:**
| ロール | 基本情報 | 傾向 | 活かし方 | SF | SPI | 評価 |
|--------|----------|------|----------|-----|-----|------|
| 管理者 | ○ | ○ | ○ | ○ | ○ | ○ |
| マネージャー | ○ | ○ | ○ | ○ | ○ | △* |
| 一般社員 | ○ | ○ | ○ | ○ | ○ | × |

*△: 自チームメンバーのみ閲覧可

## 3. ストレングスファインダー分析

### 3.1 全社分析

**画面パス:** `/analysis/strengths`

**分析ビュー:**

#### 4領域分布
- 全社員のTOP5資質を4領域に分類
- 円グラフまたは棒グラフで可視化
- 領域ごとの人数・割合表示

#### 資質ランキング
- TOP5に含まれる資質の頻度順ランキング
- 上位10資質、下位10資質を表示
- 資質名クリックで該当社員一覧表示

#### チーム別比較
- チームごとの4領域分布を並列表示
- チーム間のバランス差異を可視化

#### 職種別分析
- 職種ごとの傾向分析
- 職種に多い資質、少ない資質の特定

## 5. AI相談機能

### 5.1 AI相談画面

**画面パス:** `/ai-consultation`

**機能概要:**
自然言語でチーム編成や人材配置について相談できるAIアシスタントです。

**主な相談内容:**
- チーム編成のアドバイス
- 人材配置の提案
- 特定社員の育成方法
- チーム間の協業促進
- プロジェクトメンバー選定

**@メンション機能:**
入力欄で `@` を入力すると、社員名のサジェストが表示されます。
メンションした社員のタレント情報がAIの回答に反映されます。

**テンプレート質問:**
- 「エンジニアチームを新設する場合、どのようなメンバー構成が最適ですか？」
- 「新規プロジェクトのリーダーに適した人材を教えてください」
- 「開発チームとデザインチームの協業を円滑にするには？」
- 「@〇〇さんの成長を支援するためのアドバイスをください」

**AIモデル:**
Claude (Anthropic) を使用

**利用制限:**
| プラン | 月間相談回数 |
|--------|--------------|
| Standard | 100回 |
| Professional | 500回 |
| Enterprise | 無制限 |

---

### 5.2 AIプロフィール生成

**機能概要:**
社員のストレングスファインダー、SPI、評価データを基に、AIが自動でプロフィールを生成します。

**生成項目:**
| 項目 | 説明 |
|------|------|
| プロフィールサマリー | 人物の総合的な特徴 |
| ワークスタイル分析 | 仕事の進め方の傾向 |
| 協働のヒント | この人と効果的に働くコツ |
| 成長提案 | キャリア発展へのアドバイス |

**再生成:**
管理者は任意のタイミングでAIプロフィールを再生成できます。

**画面パス:** `/admin/evaluations`

**評価項目:**
| 項目 | 説明 |

## 8. 管理者機能

### 8.1 社員管理

**画面パス:** `/admin/employees`

**機能:**
- 社員の新規登録
- 社員情報の編集
- 退職処理（論理削除）
- 権限変更

---

### 8.2 ストレングスファインダー入力

**画面パス:** `/admin/strengths`

**機能:**
- 34資質の順位をドラッグ&ドロップで入力
- CSVインポート（一括登録）
- 入力済み社員の確認

---

### 8.3 SPI入力

**画面パス:** `/admin/spi`

**機能:**
- 性格特性スコアの入力
- ワークスタイルスコアの入力
- 適性スコアの入力
- テスト実施日の記録

---

### 8.4 チーム管理

**画面パス:** `/admin/teams`

**機能:**
- チームの作成
- チーム情報の編集
- マネージャーの設定
- チームの削除

---

### 8.5 AIプロフィール再生成

**画面パス:** `/admin/ai-regenerate`

**機能:**
- 個別社員のAIプロフィール再生成
- 一括再生成（全社員または選択社員）
- 生成履歴の確認
