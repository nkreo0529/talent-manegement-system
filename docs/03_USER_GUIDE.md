## 1. ログイン

### 1.1 ログイン画面にアクセス

ブラウザでシステムのURLにアクセスすると、ログイン画面が表示されます。

### 1.2 認証情報の入力

1. **メールアドレス**: 会社から付与されたメールアドレスを入力
2. **パスワード**: 設定したパスワードを入力
3. 「ログイン」ボタンをクリック

### 1.3 パスワードを忘れた場合

<!-- TODO(未完了): パスワードリセット機能は現在未実装。Better Auth にパスワードリセットフローを追加する必要があります -->

1. 「パスワードをお忘れですか？」リンクをクリック
2. メールアドレスを入力
3. 届いたメールのリンクから新しいパスワードを設定

## 2. ダッシュボード

ログイン後、最初に表示される画面です。

### 2.1 画面構成

```
┌─────────────────────────────────────────────────────┐
│  ヘッダー（検索バー、通知、ダークモード切替）          │
├─────────┬───────────────────────────────────────────┤
│         │                                           │
│ サイド  │   ウェルカムメッセージ                      │
│ バー    │   ───────────────────                      │
│         │   統計カード（社員数、チーム数など）        │
│         │   ───────────────────                      │
│         │   クイックアクション  │  最近のアクティビティ│
│         │                                           │
└─────────┴───────────────────────────────────────────┘
```

### 2.2 統計カード

| カード | 説明 | クリック時 |
|--------|------|-----------|
| 総社員数 | 登録されている社員の数 | 社員一覧へ移動 |
| チーム数 | 登録されているチームの数 | チーム一覧へ移動 |
| 今月の1on1 | 当月実施された1on1の数 | 評価管理へ移動 |
| AI分析済み | AIプロフィールが生成済みの社員数 | AI相談へ移動 |

### 2.3 クイックアクション

よく使う機能にすばやくアクセスできます。

- **社員を検索**: 社員一覧画面へ
- **SF分析を見る**: ストレングス分析画面へ
- **AI相談**: AI相談画面へ
- **チーム管理**: チーム管理画面へ（管理者のみ）

## 3. 社員を検索する

### 3.1 社員一覧画面を開く

サイドバーの「社員」をクリック、または検索バーに入力を開始します。

### 3.2 キーワード検索

画面上部の検索ボックスに名前の一部を入力すると、リアルタイムで絞り込まれます。

**検索対象:**
- 氏名（漢字）
- 氏名（ふりがな）
- メールアドレス

### 3.3 フィルタリング

検索ボックスの横にあるフィルタを使って絞り込めます。

| フィルタ | 選択肢 |
|----------|--------|
| チーム | 全チーム / 開発 / デザイン / 営業 など |
| 職種 | 全職種 / エンジニア / デザイナー / PM など |

### 3.4 表示切替

- **カード表示**: 顔写真付きのカード形式（デフォルト）
- **リスト表示**: コンパクトな一覧形式

## 4. 社員の詳細を見る

### 4.1 詳細画面を開く

社員一覧で社員のカードをクリックすると、詳細画面が開きます。

### 4.2 タブの切り替え

画面上部のタブをクリックして、表示内容を切り替えます。

#### 基本情報タブ
社員の基本的なプロフィール情報を確認できます。

#### 傾向タブ
ストレングスファインダーとSPIの結果から、この社員の特徴を解説します。

- **得意なこと**: TOP5資質から推測される強み
- **苦手かもしれないこと**: 下位資質から注意すべき点

#### この人の活かし方タブ
AIが生成した、この社員と効果的に働くためのアドバイスです。

- **プロフィールサマリー**: 人物の全体像
- **ワークスタイル**: 仕事の進め方の特徴
- **協働のヒント**: 一緒に働く際のコツ
- **成長提案**: 今後のキャリア発展に向けて

#### ストレングスファインダータブ
34資質の順位と、4領域での分布を確認できます。

- **TOP5**: 上位5資質を強調表示
- **4領域グラフ**: 実行力・影響力・人間関係・戦略のバランス
- **資質一覧**: 34資質の完全な順位

#### SPIタブ
SPI診断の結果をグラフで確認できます。

- **性格特性**: Big Fiveの5指標
- **ワークスタイル**: 6つの仕事傾向
- **適性スコア**: 3つの能力指標

#### 評価タブ（権限が必要）
過去の評価履歴を確認できます。

> ⚠️ このタブは、管理者または該当社員のマネージャーのみ閲覧可能です。

## 5. チーム情報を見る

### 5.1 チーム一覧

サイドバーの「チーム」をクリックすると、チーム一覧が表示されます。

各チームカードには以下が表示されます：
- チーム名
- 説明
- マネージャー名
- メンバー数
- 4領域のミニグラフ

### 5.2 チーム詳細

チームカードをクリックすると、詳細画面が開きます。

#### メンバー一覧
所属するメンバーがカード形式で表示されます。
各カードにはTOP5資質がバッジで表示されています。

#### チーム分析
チーム全体のストレングス分布を分析します。

- **4領域分布**: 実行力・影響力・人間関係・戦略のバランス
- **主要資質**: チーム内で多く見られる資質
- **補完候補**: 不足している資質を持つ人材の提案

#### AIチーム分析
AIによるチームの分析結果です。

- **チームダイナミクス**: チームの特徴と強み
- **潜在的課題**: 注意すべきポイント
- **改善提案**: チーム力向上のアドバイス

## 6. 分析レポートを見る

### 6.1 ストレングス分析

サイドバーの「分析」→「ストレングス」をクリック。

#### 全社の4領域分布
全社員のTOP5資質を4領域に分類した分布グラフ。
どの領域が強いか/弱いかが一目でわかります。

#### 資質ランキング
TOP5に含まれる資質の頻度ランキング。

- **上位資質**: 組織に多い強み
- **希少資質**: 持っている人が少ない資質

資質名をクリックすると、その資質を持つ社員の一覧が表示されます。

#### チーム別比較
チームごとの4領域分布を並べて比較できます。
チーム間のバランスの違いを把握するのに役立ちます。

### 6.2 SPI分析

サイドバーの「分析」→「SPI」をクリック。

#### 全社平均
全社員のSPIスコアの平均値をグラフで表示。

#### チーム別傾向
チームごとのSPI傾向を比較。

#### 職種別傾向
職種ごとのSPI傾向を比較。

## 7. AIに相談する

### 7.1 AI相談画面を開く

サイドバーの「AI相談」をクリック。

### 7.2 質問を入力する

画面下部の入力欄に、相談したい内容を入力します。

**相談例:**
- 「新規プロジェクトのリーダーに適した人材は誰ですか？」
- 「エンジニアチームの強みと課題を教えてください」
- 「営業チームとマーケティングチームの協業を改善するには？」

### 7.3 社員をメンションする

特定の社員について相談したい場合は、`@` に続けて名前を入力します。

1. 入力欄に `@` を入力
2. 候補が表示されるので、該当する社員をクリック
3. 質問を続けて入力

**例:**
「@田中太郎 さんの強みを活かせるプロジェクトは？」

### 7.4 テンプレートを使う

画面右側のテンプレートをクリックすると、よくある質問が自動入力されます。

| テンプレート | 質問内容 |
|--------------|----------|
| チーム編成 | 最適なメンバー構成の相談 |
| 人材配置 | リーダー候補の探索 |
| チーム相性 | チーム間協業の改善 |
| 育成 | 特定社員の成長支援 |

### 7.5 回答を読む

AIの回答は、Markdown形式で整形されて表示されます。

- **太字**: 重要なポイント
- **リスト**: 複数の提案や選択肢
- **見出し**: 回答のセクション分け

## 8. ダークモードの切り替え

### 8.1 切り替え方法

ヘッダー右側の月/太陽アイコンをクリックすると、ライトモード/ダークモードが切り替わります。

- ☀️ → ダークモードに切り替え
- 🌙 → ライトモードに切り替え

### 8.2 設定の保存

選択したモードは自動的に保存され、次回ログイン時も維持されます。

## 9. よくある質問

### Q: 自分のストレングスファインダーの結果を登録したい

A: ストレングスファインダーの結果は管理者が登録します。
人事担当者に診断結果を提出してください。

### Q: 他の社員の評価を見ることはできますか？

A: 評価情報は権限によってアクセスが制限されています。
- 一般社員: 自分の評価のみ閲覧可能
- マネージャー: 自チームメンバーの評価を閲覧可能
- 管理者: 全社員の評価を閲覧可能

### Q: AIの回答は保存されますか？

A: 現在のセッション中は会話履歴が保持されますが、
ページを離れると履歴はクリアされます。

### Q: 検索しても社員が見つからない

A: 以下を確認してください。
1. 名前のスペルが正しいか
2. フィルタが適用されていないか（「全チーム」「全職種」に変更）
3. 退職済みの社員は「在籍中」フィルタで除外されている場合があります

### Q: 表示がおかしい・動作が遅い

A: 以下をお試しください。
1. ブラウザを最新版にアップデート
2. キャッシュをクリア（Ctrl+Shift+R または Cmd+Shift+R）
3. 別のブラウザで試す

問題が解決しない場合は、サポートにお問い合わせください。
