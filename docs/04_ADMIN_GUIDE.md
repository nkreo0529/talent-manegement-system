## 1. 管理者の役割

### 1.1 権限レベル

TalentSyncには3つの権限レベルがあります。

| 権限 | 説明 | 対象者 |
|------|------|--------|
| **admin**（管理者） | すべての機能にアクセス可能 | 人事部門、システム管理者 |
| **manager**（マネージャー） | 自チームの評価・1on1を管理 | 各チームのマネージャー |
| **member**（一般） | 閲覧中心、自分の情報編集 | 一般社員 |

### 1.2 管理者ができること

- 社員の追加・編集・削除
- ストレングスファインダー結果の登録
- SPI結果の登録
- 全社員の評価閲覧・編集
- チームの作成・編集
- AIプロフィールの再生成
- 権限の付与・変更

## 2. 社員管理

### 2.1 社員管理画面を開く

サイドバー「管理」→「社員管理」をクリック

### 2.2 社員を追加する

1. 「社員を追加」ボタンをクリック
2. 必要情報を入力：

| 項目 | 必須 | 説明 |
|------|------|------|
| 氏名 | ○ | フルネーム（漢字） |
| 氏名（かな） | | ふりがな |
| メールアドレス | ○ | ログインに使用 |
| 所属チーム | ○ | プルダウンから選択 |
| 役職 | | 肩書き |
| 職種 | ○ | エンジニア、営業など |
| 入社日 | | カレンダーから選択 |
| 権限 | ○ | admin/manager/member |

3. 「保存」をクリック

### 2.3 社員情報を編集する

1. 社員一覧から対象者の「編集」アイコン（✏️）をクリック
2. 情報を修正
3. 「保存」をクリック

### 2.4 社員を退職処理する

1. 社員一覧から対象者の「削除」アイコン（🗑️）をクリック
2. 確認ダイアログで「退職処理」をクリック

> ⚠️ **注意**: 退職処理は論理削除です。データは保持されますが、一覧に表示されなくなります。

### 2.5 CSVインポート

1. 「CSVインポート」ボタンをクリック
2. テンプレートをダウンロード
3. CSVファイルを作成
4. ファイルをアップロード
5. プレビューを確認して「インポート」

**CSVフォーマット:**
```csv
email,name,name_kana,team_id,job_title,job_type,hire_date,role
tanaka@example.com,田中太郎,たなかたろう,team-1,エンジニア,engineer,2020-04-01,member
```

## 3. ストレングスファインダー入力

### 3.1 入力画面を開く

サイドバー「管理」→「SF入力」をクリック

### 3.2 社員を選択する

1. 「社員を選択」プルダウンから対象者を選ぶ
2. 既存データがある場合は、現在の順位が表示される

### 3.3 資質の順位を入力する

**方法1: ドラッグ&ドロップ**
1. 34資質が一覧表示される
2. 資質カードをドラッグして正しい順位に並べ替え
3. 上から順に1位〜34位

**方法2: TOP5のみ入力**
1. 「TOP5のみ入力」モードに切り替え
2. 1位〜5位を選択
3. 6位以下は自動でランダム配置

### 3.4 保存する

1. 順位を確認
2. 「保存」ボタンをクリック
3. 成功メッセージを確認

### 3.5 一括インポート

1. 「CSVインポート」ボタンをクリック
2. テンプレートをダウンロード
3. CSVを作成（社員ID、34資質の順位）
4. アップロードして確認
5. 「インポート」をクリック

## 4. SPI入力

### 4.1 入力画面を開く

サイドバー「管理」→「SPI入力」をクリック

### 4.2 社員を選択する

「社員を選択」プルダウンから対象者を選ぶ

### 4.3 スコアを入力する

#### 性格特性（Big Five）
各項目を1〜10のスライダーで入力：

| 項目 | 低スコア | 高スコア |
|------|----------|----------|
| 外向性 | 内向的 | 外向的 |
| 協調性 | 独立的 | 協調的 |
| 誠実性 | 柔軟 | 計画的 |
| 神経症的傾向 | 安定 | 敏感 |
| 開放性 | 保守的 | 革新的 |

#### ワークスタイル
各項目を1〜10のスライダーで入力：

- リーダーシップ
- 独立性
- チームワーク
- 粘り強さ
- 柔軟性
- ストレス耐性

#### 適性スコア
各項目を1〜10で入力：

- 言語能力
- 数理能力
- 論理能力

#### テスト実施日
SPIを実施した日付を選択

### 4.4 保存する

すべて入力後、「保存」をクリック

## 5. 評価・1on1管理

### 5.1 評価管理画面を開く

サイドバー「管理」→「評価管理」をクリック

### 5.2 評価を登録する

1. 「新規評価」ボタンをクリック
2. 情報を入力：

| 項目 | 説明 |
|------|------|
| 対象社員 | 評価対象者を選択 |
| 評価期間 | 2024H1、2024H2 など |
| 総合評価 | S/A/B/C/D から選択 |
| 強みコメント | 良かった点を記入 |
| 改善コメント | 課題・期待を記入 |
| 目標 | 次期の目標を記入 |

3. 「保存」をクリック

### 5.3 1on1を記録する

1. 「新規1on1」ボタンをクリック
2. 情報を入力：

| 項目 | 説明 |
|------|------|
| 対象社員 | 面談相手を選択 |
| 実施日 | 面談日を選択 |
| トピック | 話し合った内容のタグを選択 |
| メモ | 詳細な記録を入力 |
| アクションアイテム | 次回までの宿題を入力 |
| ムード | 本人の状態を1-5で選択 |

3. 「保存」をクリック

### 5.4 評価履歴を確認する

1. 社員名で検索
2. 期間でフィルタ
3. 評価推移グラフを確認

## 6. チーム管理

### 6.1 チーム管理画面を開く

サイドバー「管理」→「チーム管理」をクリック

### 6.2 チームを作成する

1. 「チームを追加」ボタンをクリック
2. 情報を入力：

| 項目 | 必須 | 説明 |
|------|------|------|
| チーム名 | ○ | チームの名称 |
| 説明 | | チームの役割・ミッション |
| マネージャー | | 責任者を選択 |

3. 「保存」をクリック

### 6.3 チームを編集する

1. チームカードの「編集」アイコンをクリック
2. 情報を修正
3. 「保存」をクリック

### 6.4 チームを削除する

1. チームカードの「削除」アイコンをクリック
2. 確認ダイアログで「削除」をクリック

> ⚠️ **注意**: 所属メンバーがいるチームは削除できません。先にメンバーを他チームに異動してください。

## 7. AIプロフィール再生成

### 7.1 再生成画面を開く

サイドバー「管理」→「AI再生成」をクリック

### 7.2 個別に再生成する

1. 「社員を選択」から対象者を選ぶ
2. 「再生成」ボタンをクリック
3. 処理完了まで待機（10-30秒）
4. 成功メッセージを確認

### 7.3 一括再生成する

1. 「一括再生成」タブを選択
2. 対象者を選択（チェックボックス）または「全員選択」
3. 「一括再生成」ボタンをクリック
4. 処理完了まで待機

> ⚠️ **注意**: 一括再生成は時間がかかります。大量のリクエストはシステム負荷を考慮して、業務時間外に実行してください。

### 7.4 再生成タイミングの目安

以下の場合に再生成を推奨します：

- ストレングスファインダーの結果を更新した
- SPI結果を更新した
- 評価が追加された
- 定期的な更新（四半期に1回程度）

## 8. 権限管理

### 8.1 権限を変更する

1. 社員管理画面を開く
2. 対象者の「編集」をクリック
3. 「権限」プルダウンを変更
4. 「保存」をクリック

### 8.2 マネージャーの設定

マネージャー権限を持つ社員は、自チームのメンバーに対して以下が可能：

- 評価の閲覧・編集
- 1on1の記録
- チーム分析の閲覧

**設定手順:**
1. 社員の権限を「manager」に変更
2. チーム管理で該当チームのマネージャーに設定

### 8.3 権限の一括変更

CSVインポート機能を使用して、複数社員の権限を一括変更できます。

## 10. トラブルシューティング

### Q: 社員がログインできない

**確認事項:**
1. メールアドレスが正しく登録されているか
2. パスワードがリセットされていないか
3. アカウントが有効か（退職処理されていないか）

**対処:**
- パスワードリセットメールを再送
- 認証システムのログを確認

### Q: AIプロフィールが生成されない

**確認事項:**
1. ストレングスファインダーのデータが登録されているか
2. APIキーが正しく設定されているか
3. API利用制限に達していないか

**対処:**
- SF結果を登録してから再生成
- API設定を確認
- 利用状況を確認

### Q: 権限を変更しても反映されない

**対処:**
1. 対象ユーザーに再ログインを依頼
2. ブラウザキャッシュをクリア

### Q: CSVインポートが失敗する

**確認事項:**
1. ファイルの文字コードがUTF-8か
2. 必須項目がすべて入力されているか
3. 日付形式が正しいか（YYYY-MM-DD）
4. チームIDが存在するか

**対処:**
- エラーメッセージを確認して修正
- テンプレートを再ダウンロードして作り直し

### Q: システムが遅い

**確認事項:**
1. ネットワーク接続状況
2. 同時接続ユーザー数
3. 大量データ処理中でないか

**対処:**
- 時間をおいて再度アクセス
- ブラウザを変更
- サポートに連絡
