# TalentSync - タレントマネジメントシステム

## 製品概要

**TalentSync**は、中小企業（30〜100名規模）向けに設計された、社員の強みを可視化し組織力を最大化するタレントマネジメントシステムです。

ストレングスファインダー®やSPI診断の結果をデータベース化し、AIによる分析・アドバイス機能と組み合わせることで、人事・マネージャーの意思決定を強力にサポートします。

---

## 解決する課題

| 課題 | TalentSyncによる解決 |
|------|---------------------|
| 社員の強みが把握できていない | SF/SPI結果を一元管理、検索可能に |
| チーム編成の根拠がない | 強み分布を可視化、AIがバランスを分析 |
| 1on1や評価の記録が散逸 | 社員ごとに履歴を蓄積、傾向を把握 |
| 人材配置に時間がかかる | AIアシスタントが候補を即座に提案 |
| 属人的な人事判断 | データに基づく客観的な意思決定 |

---

## 主要機能

### 1. 社員情報管理
- 基本情報、所属チーム、職種の一元管理
- 顔写真付きプロフィール
- 入社日、役職の履歴管理

### 2. ストレングスファインダー® 分析
- 34資質の順位を登録・管理
- TOP5資質をバッジ表示
- 4領域（実行力・影響力・人間関係構築力・戦略的思考力）での分析
- 全社・チーム・職種別の分布グラフ

### 3. SPI診断結果管理
- 性格特性（Big Five）の可視化
- ワークスタイル傾向の分析
- 適性スコアの記録

### 4. AI相談アシスタント
- 自然言語でのチーム編成相談
- @メンションで社員データを参照
- 個別の育成アドバイス生成
- AIプロフィール自動生成

### 5. 評価・1on1管理
- 期別評価の記録
- 1on1ミーティングのメモ・アクションアイテム
- 評価傾向の分析

### 6. チーム分析
- チーム構成と強み分布
- チームダイナミクスのAI分析
- 補完関係の可視化
